# Cybersecurity Awareness Bot 🔐

## Project Overview

The Cybersecurity Awareness Bot is a GUI-based chatbot developed in C# using Windows Forms. The chatbot helps educate users about cybersecurity awareness and online safety through interactive conversations.

---

## Features

- GUI chatbot interface
- Voice greeting using WAV file
- ASCII art cybersecurity banner
- Memory recall system
- Sentiment detection
- Random chatbot responses
- Conversation continuity
- Typing simulation
- Keyword recognition
- Input validation
- Clear chat functionality

---

## Cybersecurity Topics Covered

- Password safety
- Phishing awareness
- Malware protection
- Online privacy
- Safe browsing

---

## Technologies Used

- C#
- Windows Forms
- .NET
- GitHub
- GitHub Actions

---

## How to Run the Program

1. Open the project in Visual Studio Code.
2. Open the terminal.
3. Run the following command:

```bash
dotnet run
```
